### ===================脚本内容相关==================

### 100活跃度怎么算的

由于忘川活跃任务太杂懒得做识别了所以默认你是每天新登录：

一系列活动点完正常是有65活跃度的

（但是不排除一些极端情况比如你好友不足5个，再比如FHLC今天抽风了有的功能没点上（欢迎提供log给作者反馈修脚本））
这时候会去每日任务界面读一下目前的活跃度

不足100的部分会列一个打本计划补满

默认刷4趟宝墟补满，不够的话会再补一趟宝墟，还不够的话在选项里开启打每日金戈再补30活跃度

（故世风云要单独切章目前还没写）

刷本不处理过期加成和猫车，对刷本翻车做了一定的处理但不保证完全完美，请保证你的队伍练度足够

计划刷满后会再进一趟每日任务，领过活跃奖励后即使没满也会视作完成

这部分可能因为各种意想不到的bug中断，由于作者不写每日任务识别所以没更好的解决方式了，需要自己手动补（或者开摆）

### 每日密令功能为什么需要手动启动
微信签到部分会抢鼠标，你也不想忙正事的时候电脑上突然弹出个微信窗口吧

另外fhlc不会读您的任何微信隐私信息，请放心

另外微信签到脚本是pywinauto写的（通过windows系统的UI路径找按键），完全没继承src的优秀状态循环，很脆弱，微信最小化到后台都会找不到窗口（）

还有官方公众号也隔三差五不往签到消息里放密令，脚本目前做不到从公众号文章里找密令

综上所述失败率很高，放进调度器只会徒增血压，欢迎帮笨比作者优化这部分（悲）
